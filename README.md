📋 Dynamic Task Manager

A simple, clipboard‑style task manager that lets you add, delete, and clear tasks with ease. The task bar stays visible while you scroll.

✨ Features: 
Add new tasks
Delete selected tasks
Clear all tasks
Sticky task bar (always visible)

🔮 Future Features: 
Strike‑through animation on click
On‑screen action messages
Save tasks automatically with localStorage
Fade‑in animation for new tasks
Checkboxes for completed tasks
🛠️ Built With: 
HTML
TailwindCSS
JavaScript
Google Fonts (Patrick Hand)

🚀 How to Use: 
Type a task into the input bar.
Click Add to add it to your list.
Click Delete, then click a task, then click Delete again to remove it.
Click Clear to remove all tasks.
